Formatting Instructions

Please save your procurement worksheet as a .csv file and adjust the column headers to use our standard names before uploading. Click the section below that matches your data format:

Type 01: Quantity and units in one column

If your worksheet lists the quantity and the unit of measure in a single column:

  • Make sure to name the product column "product".
  • Change the quantity column's name to "product_weight".
  • Type "type01" anywhere in the first column (like the row just beneath your last item).
product product_weight
Harissa Paste (Tunisian) 40z EA JAR 80.000 OZ
Type 02: Total weight without a specific unit column

If your worksheet shows the total weight in pounds, but lacks a dedicated column specifying that the unit is pounds:

  • Update the item description header to say "product".
  • Update the total weight header to say "product_weight".
  • Change the quantity shipped header to "qty".
  • Insert a blank column and name its header "unit".
  • Delete any summary or "Total" rows at the bottom of the list.
  • Type "type02" anywhere in the first column (like the row just beneath your last item).
product Item # qty product_weight unit
MILK WHOLE $4/1$ GAL *RG* 77820 1 32
Type 03: Weight integrated into description

If your worksheet includes the product weight information directly within the product description field:

  • Modify the item description header to be "product".
  • Modify the quantity sold header to be "qty".
  • Delete the "Grand Total" line at the bottom if there is one.
  • Type "type03" into the first column, below the last data row.
product Stock Unit qty
CHEDDAR CLOTHBOUND AGED CABOT 30 LB MC LB 19
